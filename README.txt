تعليمات تشغيل المشروع:

1. فك الضغط عن الملف.
2. افتح Terminal أو CMD.
3. ادخل على الفولدر:
   cd accent_detector_project
4. فعل البيئة الافتراضية (اختياري):
   python -m venv venv
   .\venv\Scripts\activate
5. ثبت المكتبات:
   pip install -r requirements.txt
6. شغل التطبيق:
   streamlit run app.py
